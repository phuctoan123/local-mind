# API

The OpenAPI UI is generated by FastAPI at `/docs` when the backend is running.

Base URL:

```text
http://localhost:8000/api/v1
```

Core endpoints:

- `POST /upload`
- `GET /documents`
- `GET /documents/{document_id}`
- `GET /documents/{document_id}/chunks`
- `DELETE /documents/{document_id}`
- `POST /chat` with `stream: false` for JSON or `stream: true` for SSE
- `POST /chunks` for retrieval-only source chunks
- `POST /sessions`
- `DELETE /sessions/{session_id}`
- `GET /health`

OpenAI-compatible endpoint:

- `POST /v1/chat/completions`

Streaming `/api/v1/chat` emits Server-Sent Events:

```text
data: {"type":"sources","sources":[...]}
data: {"type":"token","content":"..."}
data: {"type":"done","latency_ms":1234}
```
